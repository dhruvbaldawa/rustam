# Task 1.1: Project Setup

## Goal

Initialize project with Vite + React + Tailwind + Firebase using standard scaffolding tools.

## Working Result

- Vite React project running locally (scaffolded via `pnpm create vite`)
- Tailwind CSS configured and working
- Firebase project created and SDK initialized
- Anonymous auth working
- Basic routing structure in place

## Constraints

- Use pnpm as package manager
- Firebase Realtime Database (not Firestore)
- Anonymous auth only

## Dependencies

None - this is the foundation task.

<guidance>

## Scaffolding Commands

```bash
# Create Vite React project
pnpm create vite rustam --template react

# Install Tailwind
pnpm add -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

# Install Firebase
pnpm add firebase

# Install React Router
pnpm add react-router-dom
```

## Firebase Setup

1. Create Firebase project at console.firebase.google.com
2. Enable Anonymous Authentication in Firebase Console > Authentication > Sign-in method
3. Create Realtime Database in Firebase Console > Realtime Database (start in test mode)
4. Register web app and copy config

## Environment Variables

Create `.env.local` with Firebase config:

```
VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=...
VITE_FIREBASE_DATABASE_URL=...
VITE_FIREBASE_PROJECT_ID=...
```
